# 🧭 Dijkstra Shortest Path Algorithm

This project demonstrates a beginner-friendly implementation of Dijkstra’s shortest path algorithm using Python. It uses an adjacency matrix to represent weighted graph connections and finds the shortest path from a starting node to other nodes.

## 📌 Objectives

- Represent a weighted graph using an adjacency matrix.
- Track shortest distances from a starting node.
- Mark visited nodes while checking graph connections.
- Reconstruct and display the shortest path between nodes.
- Practice graph-based problem solving using Python.

## 🛠️ Tools & Concepts Used

- Python
- Graphs
- Weighted Edges
- Adjacency Matrix
- Lists
- Loops
- Conditional Statements
